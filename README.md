# 🐍 贪吃蛇游戏

一个使用纯 HTML、CSS 和 JavaScript 开发的经典贪吃蛇游戏。

## ✨ 特性

- 🎮 流畅的游戏体验
- 🎨 精美的渐变 UI 设计
- 📊 实时得分和最高分记录
- 💾 本地存储最高分
- 🎯 网格化游戏界面
- ⌨️ 支持方向键和 WASD 控制
- 📱 响应式设计

## 🎯 游戏规则

1. 使用方向键（↑↓←→）或 WASD 键控制蛇的移动方向
2. 吃到红色食物可以得分，蛇身会变长
3. 不要撞到墙壁或自己的身体
4. 挑战更高的分数！

## 🚀 如何开始

直接在浏览器中打开 `index.html` 文件即可开始游戏。

或者访问 GitHub Pages：[在线试玩](https://missolin.github.io/snake-game/)

## 🎮 控制方式

- **方向键** ↑↓←→ 或 **WASD** 键控制蛇的移动
- **开始游戏** 按钮开始游戏
- **重新开始** 按钮重置游戏

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3 (渐变、动画、响应式设计)
- 原生 JavaScript (ES6+)
- LocalStorage API
